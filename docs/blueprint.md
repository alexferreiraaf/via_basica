# **App Name**: Gospel Viva Store

## Core Features:

- Product Catalog: Display a catalog of products, including books, bibles, and gifts.
- Shopping Cart: Allow users to add products to a shopping cart and manage quantities.
- WhatsApp Order: Enable users to send their order to the store via WhatsApp.
- Admin Product Management: Admin interface to add, edit, and delete products.
- Store Configuration: Admin interface to configure store details such as name and phone number.
- Product Image Generation: If no product image is provided by the admin, use generative AI tool to create a representative image based on the product title.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to convey trust and spirituality.
- Background color: Very light blue (#F0F9FF) for a clean and calming backdrop.
- Accent color: Purple (#7E22CE) to highlight key actions and elements.
- Headline font: 'Playfair', serif, for an elegant, high-end feel.
- Body font: 'PT Sans', sans-serif, as a supporting font to Playfair.
- Use 'lucide-react' icons for a clean and consistent look.
- Maintain a clean, grid-based layout with ample spacing to improve readability and visual appeal.
